# DocMetaAI

A simple document assistant that can scan Word documents, provide AI-style suggestions, fix formatting, and export PDF.

## Run locally

```bash
python -m streamlit run app.py
```

## Deploy publicly

The easiest public deployment is Streamlit Community Cloud.

1. Push this project to GitHub.
2. Open https://share.streamlit.io/
3. Click New app.
4. Choose the GitHub repository and the file `app.py`.
5. Deploy.

Once deployed, Streamlit will give you a public URL that anyone can open.
